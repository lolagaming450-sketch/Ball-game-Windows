1. Save the File
  Open the Notepad application on Windows.
  Paste the Windows version of the Python code into the document.
  Select File > Save As.
  Change the "Save as type" dropdown menu from Text Documents (.txt)* to All Files (.).
  Name the file exactly multiplier.py and save it directly to your Desktop.
2. Open Command Prompt
  Press the Windows Key, type cmd, and press Enter to open the Command Prompt.
  Change your directory to the Desktop folder by running the following command: "cd Desktop"
3. Install the Requirements
  Windows installs the necessary graphics and audio libraries using the pip package manager. Run this command in the     Command Prompt: "pip install PyQt6 pygame"
  Note: If the system returns a "'pip' is not recognized" or "'python' is not recognized" error, Python is not           installed yet. Download and run the official installer from python.org, making sure to check the box that says "Add    python.exe to PATH" during installation.
4. Run the Script
  To start the transparent overlay, type this final command into the Command Prompt and press Enter: "python             multiplier.py".
